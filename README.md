This repository contains the Week 2 project for the TIQC internship. It is a modern web application built with Next.js, bootstrapped with the App Router, and styled using Tailwind CSS and shadcn/ui.

## Tech Stack

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **UI Components:** [shadcn/ui](https://ui.shadcn.com/)
* **Package Manager:** [pnpm](https://pnpm.io/)

## Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine. This project uses `pnpm` as its package manager. If you don't have it installed, you can install it globally via npm:

```bash
npm install -g pnpm
```

## Installation

1. Clone the repository: git clone [https://github.com/Lenfried/TIQC-internship-week2-Chen.git](https://github.com/Lenfried/TIQC-internship-week2-Chen.git)
2. Navigate into the directory: cd TIQC-internship-week2-Chen
3. Install the dependencies: pnpm install
4. Run the server: pnpm dev
5. Open http://localhost:3000 in your browser to interact with the application.
